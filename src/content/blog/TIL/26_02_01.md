# 26-02-01
When you build like a open world game or application and you want to avoid that the user leaves a certain area, for example where you have some floor plane, you can analyse the texture if you do see any floor texture or not. If not return back to previous milestone.

Also if you want to know where you are between different milestones you can calculate the distance to the next or last milestone.
