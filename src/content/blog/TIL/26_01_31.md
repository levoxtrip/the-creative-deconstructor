To find all operators of a type with a specific name in a COMP you can use
`me.parent().findChildren(name='feedback*',type=TOP)`


# TD

![Initial Values on Start Img](/img/TD/InitialValuesOnStart.png)

To set initial values on startup you can use a constant chop with all the initial values and then connect it to a `overrideCHOP` the second input then overwrites your initial values when they change.
