# 26-05-11
If you want to access a website that you host on your local machine from like your phone for example, you have to be in the same wifi. Host a http server with python. Then access the website from your other device with the ip from the PC + the name of the html file like `index.html`. To connect websocket add `ws:// ip + port`. In windows defender create a rule for the port so it is not blocked.
