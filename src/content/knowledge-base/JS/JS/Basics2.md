# Javascript

## Filter
`.filter(item => item.name === searchedName)` takes an array and creates a new array with only the items that match a condition.

It goes through all the elements of an array and for each article it runs a function to check should i keep this one or not

## Find
To find one specific item in an array which matches the condition you can use `.find()`
