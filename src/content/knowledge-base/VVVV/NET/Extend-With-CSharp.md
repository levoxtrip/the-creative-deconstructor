# Extend Patch With C# 
When you want to create your own nodes with C# or want to extend your patch with C# code you can go to `New->C#File` and click on `Create`. This then opens a the C# project for your project. Inside it you find the script where you can add your C# functionality.


![Create C# Img](/img/VVVV/CreateCSharpFile.png).

More important information [here in the VVVV graybook](https://thegraybook.vvvv.org/reference/extending/writing-nodes.html#examples)
