# Best Practices

## Building a Project
It can be useful to create certain functionality in an extra level so you don't need to deal with the whole logic of your level just for play testing. 

## Store information of your game
The `Game Mode` is a good place to keep track of the rules of your game, how to win, how to loose. 

If you have a game where you for example collect objects you want to store the information in the game mode as well. You can program that logic into the event graph of the game mode.
