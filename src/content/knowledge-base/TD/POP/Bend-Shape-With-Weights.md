# Bend Shape With Weights
![Bend Shape With Weights Img](/img/TD/Bend-Shape-With-Weights.png)
We can create a `Weight Attribute` that we can use to define how strong certain transformations get applied to our shape. In this example I used a `NormalizePOP` to create the `weight` attribute. In the `TransformPOP` we then use that attribute to define how the transformations affect our shape.
