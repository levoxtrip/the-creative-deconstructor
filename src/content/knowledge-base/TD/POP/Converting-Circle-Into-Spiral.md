# Converting CirclePOP Into Spiral
![Bend Shape With Weights Img](/img/TD/ConvertingCircleIntoSpiral.png)
In a `MathMixPOP` we can use the build-in attributes point index `_PointI` or normalized point index `_PointU` to create spirals from a circle source shape by offsetting points by a scaler and the attributes.
