# Define Field Parameter Via Attributes
We can use some data from for example a `PointPOP` to create a per point field and also define different parameters for each point. For that use `PointPOP` create attributes that have the same name as the parameters of the `FieldPOP` for example `radx`,`rady` or `mode`. Then in the `FieldPOP` assign the operator in the `Specification POP`.
