# Reference Parameter Via Table
If you have a lot of reference that you have to assign you can use a `tableDAT` to assign which value you want to reference to which parameter. The `path` is the reference to the operator. The `parameter` is the parameter you want to assign the value to. The `value` is the reference to where the value is and `enable` is setting the reference to true.
